줄바꿈-br,p


1.br을 하면 한줄이 바뀜(중복사용가능) br은 닫히는 태그가 없다.
2.단락을 바꾸는 태그p,/p p태그가 더 좋다. 같은 단락이라는 것을 의미로 표현할수있다. br태그는 그냥 줄바꿈 하지만 P태그는 여러줄을 뗄수가없다.
3.CSS(언어)를 사용하면 여러줄을 뗄수있다. CSS는 html과 다른 언어(여기서는 잊어도됨)

    문서의 구조와 슈퍼스타들 - title, meta charset, head, body, html, !doctype html

    title : 웹페이지의 제목을 설정할 때 사용
    기본적으로 html은 utf-8 방식으로 저장된다. 따라서 웹페이지에서도 utf-8 방식으로 불러들여야 오류(글자가 깨짐)가 안남
    그래서 html 앞 단에 meta charset="utf-8" 을 써줘서 utf-8 방식으로 문서를 읽도록 한다.
    head, body : 본문(body)과 본문을 설명하는 태그(head)로 나뉨, 이것은 사람간의 약속
    head, body 보다 상위에 html이 있고 관용적으로 문서의 맨위에는 !doctype html을 사용한다.(이건 닫히는 태그없음)




    html 태그의 제왕 - a(링크)

    a : anchor 의 약자, 링크를 만들어 줌 네이버
    href : hypertext reference의 약자, 링크의 주소를 입력
    target 이후 : 새로운 탭으로 연다는 뜻
    title : hypertext에 갖다 댔을 때 설명이 뜨게함
